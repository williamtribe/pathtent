# 개인정보처리방침

**최종 업데이트: 2026년 1월 21일**

## 1. 개요

Patent Guide Assistant("본 확장 프로그램")는 특허청(patent.go.kr) 웹사이트에서 특허 고객 등록 절차를 안내하는 Chrome 확장 프로그램입니다.

## 2. 수집하는 정보

본 확장 프로그램은 **어떠한 개인정보도 수집하지 않습니다**.

### 로컬 저장 데이터

다음 데이터는 **사용자의 기기에만 저장**되며, 외부로 전송되지 않습니다:

- **가이드 진행 단계**: 사용자가 현재 어느 단계에 있는지를 나타내는 숫자 (예: 1, 2, 3...)
- 저장 위치: Chrome Local Storage (사용자 기기)
- 저장 목적: 페이지 이동 후에도 가이드를 이어서 진행할 수 있도록 함

## 3. 데이터 사용

본 확장 프로그램은:

- ✅ 사용자 기기에만 데이터 저장 (로컬)
- ❌ 외부 서버로 데이터 전송 안 함
- ❌ 개인정보 수집 안 함
- ❌ 쿠키 사용 안 함
- ❌ 제3자와 데이터 공유 안 함
- ❌ 분석 도구 사용 안 함

## 4. 권한 사용

본 확장 프로그램이 요청하는 권한과 사용 목적:

| 권한 | 사용 목적 |
|------|----------|
| `activeTab` | 현재 탭의 URL 확인 (올바른 가이드 단계 표시) |
| `storage` | 가이드 진행 단계를 로컬에 저장 (개인정보 저장 안 함) |
| `scripting` | patent.go.kr 페이지에 가이드 UI 표시 |
| `host_permissions` (patent.go.kr) | 특허청 사이트에서만 작동 |

## 5. 데이터 보안

- 모든 데이터는 사용자 기기에만 저장됩니다
- 외부 서버나 클라우드에 데이터를 전송하지 않습니다
- 제3자가 데이터에 접근할 수 없습니다

## 6. 데이터 삭제

사용자는 언제든지 다음 방법으로 저장된 데이터를 삭제할 수 있습니다:

1. 확장 프로그램 제거
2. Chrome 설정 > 개인정보 및 보안 > 인터넷 사용 기록 삭제
3. 확장 프로그램의 "처음부터 다시" 버튼 클릭

## 7. 어린이 개인정보 보호

본 확장 프로그램은 어떠한 연령의 사용자로부터도 개인정보를 수집하지 않습니다.
