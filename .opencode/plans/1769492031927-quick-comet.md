## Implementation Steps

### Step 1: Backend - Remove Unused Dependencies
```bash
cd projects/backend
```
**pyproject.toml에서 제거:**
- `"openpyxl>=3.1.5",` (line 13)
- `"pandas>=2.2.0",` (line 29)
- `"kiwipiepy>=0.21.0",` (line 31)

### Step 2: Backend - Sync & Verify
```bash
uv sync
uv run basedpyright
uv run pytest tests/unit/ -v
```

### Step 3: Frontend - Remove Unused Dependencies
```bash
cd projects/frontend-landing
npm uninstall @fontsource-variable/mona-sans @gsap/react @radix-ui/themes heroicons material-symbols
```

### Step 4: Frontend - Verify
```bash
npm run type-check
npm run build
```

### Step 5: Commit & Deploy
```bash
git add -A
git commit -m "chore: remove unused dependencies (backend & frontend)"
git push
```
